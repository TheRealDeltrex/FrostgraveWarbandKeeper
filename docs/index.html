❄️ Frostgrave Warband Keeper

A local warband tracker for Frostgrave (2nd Edition). No login, no server, no cloud — everything runs on your own PC and your warbands are saved as plain files on your machine.

Not affiliated with Osprey Games / Joseph A. McCullough.

🪟 Windows

No Python install needed.

Download for Windows
  1. Unzip it anywhere (e.g. your Desktop or Documents).
  2. Run FrostgraveWarbandKeeper.exe inside the unzipped folder.
  3. It opens in your browser automatically. A tray icon lets you quit it later.

Windows may show a SmartScreen warning first time (unsigned exe) — click "More info" → "Run anyway".

🐧 Linux

No Python install needed.

Download for Linux
  1. Extract it: tar -xzf FrostgraveWarbandKeeper-linux-x64.tar.gz
  2. Make it executable: chmod +x FrostgraveWarbandKeeper
  3. Run it: ./FrostgraveWarbandKeeper — opens in your browser automatically.

Or from a file manager: right-click → Properties → allow executing, then double-click.

What it does

  • Create a warband: wizard, spells (per 2e rules), an optional apprentice, and starting soldiers.
  • Between games: level up, record post-game loot, manage the vault.
  • Soldiers: hire from the full 2e roster, track status, reorder, optionally level up.
  • Captain homerule: an optional, fully-tunable house rule for hiring or promoting a Captain.
  • Home base, reference pages (spells, arms & armour, bestiary), and a clean PDF roster export.
